Working with Node.JS file system

File system API

In order to work with fs API over TypeScript you have to add @types/node library:

yarn add @types/node

Notable functions

                        
                            fs.writeFile(file, data[, options], callback);
                            fs.readFile(path[, options], callback);
                            fs.read(fd, buffer, offset, length, position, callback);
                            fs.lstat(path, callback);

And much more

Sync API

Some of the API are available as sync

                        
                            fs.writeFileSync(file, data[, options]);
                            fs.lstatSync(path);
                            fs.existsSync(path);
                            fs.readFileSync(path[, options]);
* When using sync function wrap with try..catch

    • Change ItemsController to use this service instead of in-memory array
    • Every item should be in separate file when the filename is: [item_name].json
    • Don't use sync version of API
