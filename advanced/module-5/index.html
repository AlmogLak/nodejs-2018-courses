GraphQL over Node.js

What is GraphQL?

  • Created by Facebook
  • Query language for APIs
  • Building robust applications based on intuitive and flexible syntax
  • Gives clients the power to ask for exactly what they need and nothing more

Developer Tools

GraphQL in Node.js

                        
                            npm i -s express graphql express-graphql mongoose
                            npm i -D @types/express @types/mongoose
                        
                    
                        
                            import Schema from "./schema";
                            const app = express();

                            mongo.connect('mongodb://nodejs_user:Aa123456@ds117200.mlab.com:17200/course-2018', {
                                useNewUrlParser: true,
                                useUnifiedTopology: true
                            });
                            mongo.connection.once('open', () => {
                                console.log('connected to database');
                            });

                            app.use('/graphql', graphqlHTTP({
                                schema: Schema,
                                graphiql: true
                            }));
                            app.listen(3000, () => {
                                console.log('Server running succefully...')
                            });
